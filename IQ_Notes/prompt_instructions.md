# Prompt Instructions for Creating Concept Notes

When I ask a conceptual or educational question, follow this workflow:

1. **Identify the concept** from my question
2. **Create a `.md` file** in the `IQ_Notes` folder with the concept as the filename (e.g., `hoisting.md` for a question about hoisting)
3. **Explain the concept** clearly and concisely, covering:
   - What it is
   - How it works
   - Key details / gotchas
   - Simple code examples (if applicable)
4. **Save it** to the `IQ_Notes` folder

**Example:**

> User: "What is hoisting in JavaScript?"
> → Creates `IQ_Notes/hoisting.md` explaining hoisting with code examples

**Trigger:** Any question starting with "explain", "what is", "how does", "understand", or similar conceptual/educational phrasing.
